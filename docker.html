docker 安装
  1. 更新 yum   yum update
  2. 设置docker仓库 安装 yum install -y yum-utils device-mapper-persistent-data lvm2
  3. 设置docker仓库源 yum-config-manager --add-repo  https://download.docker.com/linux/centos/docker-ce.repo
  4. 开始安装  yum install docker-ce

docker login 登录之后就可以开始拉取镜像

docker run -it node:latest /bin/bash

docker run -d -p 8080:80 --name myXXXX nginx  运行一个名字为 myXXXX 的 nginx 容器

docker exec -it xxxxxx(containerId) /bin/bash(路径) 进入容器目录执行命令

通过 docker pull 拉取的 镜像可能没有安装vim 可以使用 1. apt-get update     2. apt-get install vim 安装vim 或者其他软件
Dockerfile生成镜像：docker build -f /path/to/a/Dockerfile . 注意后面的.

docker 运行nginx   nginx 的镜像需要nginx.conf  和 配置目录的 多个server  config文件
-----运行命令
##web  
docker run \
-d -p 80:80 \
-v /usr/local/PermissionManage:/etc/PermissionManage \
-v /usr/local/nginxconf/nginx.conf:/etc/nginx/nginx.conf \
-v /usr/local/nginxconf/conf:/etc/nginx/conf.d \
--name nginx1 \
nginx

##service
docker run \
-d -p 81:3000 \
-v /root/files/node-api:/home/api/v1 \
--name nodeapi1 \
nodeapi
